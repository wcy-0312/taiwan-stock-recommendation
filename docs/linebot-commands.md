# LINE Bot 指令說明

## 問候

| 輸入 | 回覆 |
|------|------|
| 你好、嗨、hi、hello、哈囉 | 親切介紹 + Quick Reply 按鈕 |

## 股票查詢

| 指令 | 說明 | 範例 |
|------|------|------|
| `查 <代號>` | 查詢個股雷達 | 查 2330 |
| `<4位數字>` | 直接輸入代號查詢 | 2330 |
| `今日雷達` | 今日播報摘要 | — |
| `強勢股` | 今日強勢觀察清單 | — |
| `轉弱股` | 今日轉弱警示清單 | — |
| `熱門股票` | 常用股票快速選單 | — |
| `可查股票` | 目前支援的股票清單 | — |

## 追蹤清單

| 指令 | 說明 | 範例 |
|------|------|------|
| `追蹤 <代號>` | 加入追蹤清單 | 追蹤 2330 |
| `移除 <代號>` | 從清單移除 | 移除 2330 |
| `我的清單` | 查看追蹤清單（含今日狀態） | — |

## 說明

| 指令 | 說明 |
|------|------|
| `幫助` / `help` / `功能` / `怎麼用` | 顯示功能卡 |

## 注意事項

- 所有指令皆為量化技術觀察，不構成投資建議
- 目前主要追蹤 台灣 0050 成分股（共 50 支）
- 每日分析約 14:00 CST 更新，推播約 08:30 CST 發送
