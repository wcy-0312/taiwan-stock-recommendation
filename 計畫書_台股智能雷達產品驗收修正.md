
### WS-5：Web 圖表 (G)
**目標：** `/stock/{code}` 頁面顯示 4 張 async Plotly 圖表。

**`/api/stock/{code}/history?days=120`（新 API endpoint）**
- 在 `app/web/routes.py` 新增
- 呼叫 `yfinance.download(ticker, period="6mo")`（不固定 120 天以避免假日計算問題）
- 計算 MA5/MA20/MA60、RSI(14)、MACD(12,26,9)
- 回傳 JSON：`{ticker, dates[], close[], open[], high[], low[], volume[], ma5[], ma20[], ma60[], rsi[], macd[], macd_signal[], macd_hist[]}`
- 若 yfinance 失敗（timeout/exception）：回傳 `{error: "無法取得歷史資料，請稍後再試"}`，HTTP 200

**`templates/stock.html`**
- 頁面載入後，JS 發 fetch `/api/stock/{code}/history?days=120`
- 4 個 `<div id="chart-*">` 先顯示 loading spinner
- 拿到資料後用 `Plotly.newPlot` 渲染：
  1. 價格走勢（K線 + MA5/MA20/MA60）
  2. 成交量
  3. RSI(14)（加 70/30 水平線）
  4. MACD（柱狀 hist + signal + macd 折線）
- 若 API 回傳 `error` 欄位：各 chart div 顯示友善錯誤訊息（不當機）
- CDN Plotly：`https://cdn.plot.ly/plotly-latest.min.js`
